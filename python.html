01 What is Python?

Python is a high-level, general-purpose programming language created by Guido van Rossum in 1991. It has become the dominant language for AI, machine learning, and data science because of its readable syntax and massive ecosystem of scientific libraries. Every Azure AI SDK — including the ones used in these labs — has a Python client.

👁
Readable syntax
Code reads almost like English. You focus on solving the problem, not fighting the language.
📦
Rich libraries
NumPy, pandas, scikit-learn, TensorFlow — all built for Python and used by AI researchers worldwide.
👥
Huge community
Millions of developers. Every Azure AI SDK — Translator, Face, OpenAI — has a Python client.
☁️
Runs anywhere
Local machine, Azure Cloud Shell, VS Code for the Web, Jupyter notebooks — Python works in all of them.
Key pattern in all lab scripts: import a library → set your credentials → call an API → print or process the result. You will see this four-step pattern in every Python client script in these labs.
02 Python Syntax Basics

Variables & data types

Python determines the data type automatically — no need to declare it.

# Python figures out the type automatically
name       = "Ada Lovelace"   # str   — text
age        = 36               # int   — whole number
confidence = 0.97             # float — decimal
is_active  = True             # bool  — True or False

# f-strings — embed variables directly inside a string
print(f"Hello, {name}! Confidence: {confidence}")
output
Hello, Ada Lovelace! Confidence: 0.97

Lists — ordered collections

languages = ["Spanish", "French", "Arabic", "Japanese"]

print(languages[0])              # Spanish  ← index starts at 0
print(len(languages))            # 4

for lang in languages:           # loop through every item
    print(f"Translating to {lang}")

Dictionaries — key-value pairs

Why dictionaries matter: The JSON responses from every Azure AI API come back as Python dictionaries. Learning to navigate them is the single most important skill for these labs.
# A dictionary maps keys to values — exactly like a JSON object
face_result = {
    "faceId": "c5c24a82-6845",
    "blurLevel": "Low",
    "confidence": 0.99
}

print(face_result["blurLevel"])     # Low
print(face_result["confidence"])    # 0.99

for key, value in face_result.items():
    print(f"{key}: {value}")
output
Low
0.99
faceId: c5c24a82-6845
blurLevel: Low
confidence: 0.99
Functions — def keyword
def check_quality(q):
    if q == "High":
        return "✓ Proceed"
    else:
        return "✗ Retry"

print(check_quality("High"))
# ✓ Proceed

quality = "Medium"

04 AI Code Examples — Lab Patterns

The four examples below mirror the exact patterns used in the Translation and Face Analysis labs. Study each one to understand how Python connects to Azure AI services.

Example 1 — Translation API call (Translator lab)

This is the exact four-step pattern used in the Translation lab: import → credentials → call API → print result.
import requests, uuid

# Step 1 — Credentials
KEY      = "your-key"
REGION   = "eastus"
ENDPOINT = "https://api.cognitive.microsofttranslator.com"

# Step 2 — Build authentication headers
headers = {
    "Ocp-Apim-Subscription-Key": KEY,
    "Ocp-Apim-Subscription-Region": REGION,
    "Content-type": "application/json",
    "X-ClientTraceId": str(uuid.uuid4())
}

# Step 3 — Call the API
body   = [{"text": "Hello, how are you?"}]
params = {"api-version": "3.0", "to": ["es", "fr"]}
response = requests.post(
    f"{ENDPOINT}/translate",
    params=params, headers=headers, json=body
)

# Step 4 — Parse JSON and print
data = response.json()
for t in data[0]["translations"]:
    print(f"[{t['to']}]  {t['text']}")
output
[es]  Hola, ¿cómo estás?
[fr]  Bonjour, comment allez-vous?

Example 2 — Language detection

texts = ["¿Dónde está la biblioteca?", "Je voudrais un café."]

body     = [{"text": t} for t in texts]   # list comprehension
response = requests.post(
    f"{ENDPOINT}/detect",
    params={"api-version": "3.0"},
    headers=headers, json=body
)

for i, r in enumerate(response.json()):
    print(f"{texts[i][:22]}... → {r['language']} ({r['score']:.0%})")
output
¿Dónde está la bib... → es (100%)
Je voudrais un ca... → fr (100%)

Example 3 — Parsing nested JSON (Face API lab)

Nested dictionaries — API responses are often deeply nested. Use chained bracket access: result["faceAttributes"]["blur"]["blurLevel"]
# Simulated Face API response
face_response = [{
    "faceRectangle": {"top": 45, "left": 194},
    "faceAttributes": {
        "blur": {"blurLevel": "Low", "value": 0.12},
        "qualityForRecognition": "High"
    }
}]

face    = face_response[0]
quality = face["faceAttributes"]["qualityForRecognition"]
blur    = face["faceAttributes"]["blur"]["blurLevel"]

print(f"Quality: {quality} | Blur: {blur}")
output
Quality: High | Blur: Low

Example 4 — Loop + condition filtering (used in both labs)

This pattern — loop through results → check a condition → take action — appears in almost every Azure AI client script in these labs.
faces = [
    {"id": 1, "quality": "High",   "yaw": 3.1},
    {"id": 2, "quality": "Low",    "yaw": 45.2},
    {"id": 3, "quality": "High",   "yaw": 8.5},
]

for face in faces:
    q = face["quality"]
    y = face["yaw"]

    if q == "High" and y < 15:
        print(f"Face {face['id']}: Proceed to identification")
    elif q == "Low":
        print(f"Face {face['id']}: Low quality — retry")
    else:
        print(f"Face {face['id']}: Reposition — yaw={y}°")
output
Face 1: Proceed to identification
Face 2: Low quality — retry
Face 3: Proceed to identification

06 How to Run Python in These Labs
1
Open VS Code for the Web — navigate to https://vscode.dev in any browser. No installation required.
2
Open the terminal — press Ctrl + ` (backtick) or go to Terminal → New Terminal.
3
Install a library — type pip install requests (or the specific package name) and press Enter.
4
Create a .py file — click New File, name it my_script.py, and paste the lab code.
5
Replace credentials — swap "your-key" and "your-endpoint" with your actual Azure resource values.
6
Run the script — type python my_script.py in the terminal and press Enter. Output appears below.
